# Retail Sales Analysis Dashboard

## 📊 Project Overview
This project analyzes retail sales data using SQL and Power BI to uncover business insights.

## 🛠 Tools Used
- SQL (MySQL)
- Power BI
- Excel

## 📈 Key Insights
- West region generates the highest sales
- Technology category leads revenue
- Higher discounts reduce profitability
- Some sub-categories show negative profit
